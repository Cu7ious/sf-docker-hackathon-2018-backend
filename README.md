## Getting Started

1. Make sure you have [NodeJS](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.
2. Install your dependencies

    ```
    cd path/to/-authority
    ./install.sh

    ```

3. Start backend service

    ```
    ./dev-server.sh -b
    ```

3. Start frontend dev server

    ```
    ./dev-server.sh -f
    ```
